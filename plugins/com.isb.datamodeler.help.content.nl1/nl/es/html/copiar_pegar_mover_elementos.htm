Copiar, Pegar y Mover elementos

Desde el Navegador Data Modeler se permiten realizar las acciones de copiar, pegar y mover tablas. A continuación se detallan estas acciones:

•      Copiar: Para acceder a esta opción sitúese sobre la tabla o tablas que quiera copiar y haga clic en el botón derecho del ratón y seleccione Copiar. Cuando se copia una única tabla, las FKs no serán copiadas. En el caso de que se copien varias tablas, y estas estén relacionadas entre sí, se copiaran las FK que relacionen dichas tablas.

 

 

•      Pegar: Estará activa siempre que:

-     Tengamos algún contenido en el portapapeles.

-     Tengamos sólo un elemento seleccionado como destino en el navegador.

-     El elemento seleccionado en el navegador sea un esquema.

Esta acción implica pegar también aquellos elementos ligados por debajo que han sido copiados junto con el elemento de destino. Cunado haya seleccionado una única tabla para copiar, las FK no se pegarán.

 

 

Si el elemento que vamos a pegar ya existe en destino se muestra una ventana que permite cambiar el nombre del elemento.

 

 

•      Mover: Estará activa si existen varios esquemas en el proyecto. Ya que si solo hay un esquema no es posible mover la tabla de sitio. Para acceder a esta opción sitúese sobre la tabla o tablas que quiera mover y haga clic en el botón derecho del ratón y seleccione Mover.

 

 

Se muestra una ventana en la que debe seleccionar el esquema destino donde se moverá la tabla. Una vez hecho esto haga clic en .

 

 

Si el elemento que vamos a mover ya existe en destino se muestra una ventana que nos permite cambiar el nombre del elemento.

 

 

Por último, en el Navegador Data Modeler se mostrará la tabla renombrada.

 

 

Las acciones de copiar y mover también se encuentran disponibles en el diagrama de modelado. Estas se localizan haciendo clic en el botón derecho del ratón dentro de la opción Editar. La acción de pegar solo está disponible al seleccionar un esquema en el Navegador Data Modeler.
