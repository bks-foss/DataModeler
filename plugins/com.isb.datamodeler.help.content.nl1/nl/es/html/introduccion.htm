Introducción

Data Modeler es una perspectiva integrada en BKS Platform que permite realizar modelos de datos empleando un modelo relacional.

Esta herramienta se crea dentro del marco de reducción del ciclo de vida de las aplicaciones, siguiendo la estrategia de integrar todas las herramientas en un mismo entorno de desarrollo.

 

Con la creación de Data Modeler para realizar los modelos de datos se consigue que:

1.    El modelo lógico BKS y el modelo de datos estén integrados en la misma plataforma.

2.    La importación de las tablas físicas del modelo de datos desde el Diccionario de datos.
