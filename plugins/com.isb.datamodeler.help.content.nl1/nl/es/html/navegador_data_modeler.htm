Navegador Data Modeler

La vista Navegador Data Modeler permite crear o editar un proyecto de modelado de datos que estará compuesto por Esquema, Diagramas, Tablas, etc. Inicialmente esta vista no contiene ningún proyecto.

Es posible arrastrar las tablas desde el Navegador Data Modeler a los diagramas.

La siguiente imagen muestra un ejemplo de un proyecto desplegado:

 

 

En el Navegador Data Modeler, si hace clic en el botón derecho del ratón, se muestran entre otras las siguientes acciones:

•     Copiar, Pegar y Mover elementos

•     Renombrar elementos

•     Importar Tablas

•     Detectar posibles Relaciones
