Comparador de modelos de datos

El comparador de modelos de datos permite comparar funcionalmente los elementos del proyecto de modelado, ya sea de local a local o de local a CVS, localizando las diferencias de los elementos de modelo, esquemas y tablas externas.

El comparador se encuentra dividido en tres secciones. En la parte superior muestra un árbol con las diferencias existentes entre las estructuras de los elementos seleccionados. Las correspondencias entre elementos se muestran por nombre y de la raíz cuelgan los distintos esquemas. Bajo los esquemas, se muestran los elementos asociados (tablas) y bajo éstas las columnas y restricciones. Adicionalmente, bajo los elementos se muestra el nodo Atributos que representa que existe una diferencia en las propiedades del elemento correspondienteee.

En la parte inferior se muestran dos secciones, una sección muestra el contenido del elemento en el primer recurso seleccionado y la otra muestra el contenido del segundo, reflejando las diferencias que existen entre ambos. Las diferencias que aparecen resaltadas son los cambios entre el elemento origen y el elemento heredado común.

La comparación de los proyectos se puede llevar a cabo de las siguientes formas:

•     Comparación de local a local

•     Comparacion de local a cvs
