Paleta del editor de Diagrama

Permite incluir los elementos arrastrándolos desde la paleta al editor. Normalmente se encuentra disponible en la parte derecha del editor, aunque es posible desplazarla a la izquierda.

Los elementos que componen la paleta son:

•      Si selecciona este elemento y posteriormente hace clic en el editor se creará un nuevo elemento de tipo tabla. Para más información consulte Cómo crear una tabla

•    Si selecciona este elemento y posteriormente hace clic dentro de una tabla, se creará un nuevo elemento columna dentro de la tabla. Para más información consulte Cómo crear una Columna

•     Si selecciona este elemento y posteriormente hace clic dentro de una tabla, se creará un nuevo elemento Clave única dentro de la tabla. Para más información consulte Cómo crear una clave única.

•     Tipo de relación que se establece entre tablas y que consiste en que la lista de columnas involucradas de la tabla hija forman parte de  su  Clave Única (PK). Para más información consulte Relaciones del Diagrama

•    Tipo de relación que se establece entre tablas en la que la lista de columnas involucradas de la tabla hija no forman parte de su Clave Única (PK). Para más información consulte Relaciones del Diagrama.

 

Además, contiene los siguientes botones:

•       Permite ocultar la paleta.

•        Permite poner el ratón en modo de selección.

•         Permite ampliar la imagen del editor. Cada vez que haga clic en el botón izquierdo del ratón, sobre el editor, la imagen será ampliada.

•        Permite reducir la imagen del editor. Cada vez que haga clic en el botón izquierdo del ratón, sobre el editor, la imagen será reducida.

•        Existen tres tipos de notas que puede vincular a ciertos elementos incluidos en el editor, estas son:

-    Nota: Permite incluir una nota simple.

-    Texto: Permite incluir una caja de texto.

-    Nota anexa: Permite incluir una nota anexa.
