Cómo utilizar DataModeler

Data Modeler es una herramienta que permite realizar modelos de datos empleando un modelo relacional.

La herramienta se compone básicamente de las siguientes vistas:

•      Navegador Data Modeler

•      Propiedades

 

Y de una serie de editores para el modelado de las aplicaciones.

La siguiente imagen muestra la perspectiva Data Modeler con un proyecto de modelado abierto.

 

 

Los elementos generados automáticamente o manualmente con Data Modeler han de seguir la nomenclatura recogida en el documento Nomenclatura Data Modeler

 

Pasos

 

1.    Inicie la perspectiva Data Modeler desde la plataforma Helios. En la barra de herramientas seleccione: Ventana>Abrir perspectiva>Otras..., y elija Data Modeler.

 

 

Se abre la perspectiva  mostrando una serie de vistas y editores que ayudarán a completar la funcionalidad de los distintos elementos de modelado de las aplicaciones.

2.   Desde la vista Navegador Data Modeler, cree un proyecto Data Modeler, o seleccione un proyecto de modelado ya existente en su espacio de trabajo. El proyecto Data Modeler contiene los esquemas y tablas del modelado de una aplicación.
