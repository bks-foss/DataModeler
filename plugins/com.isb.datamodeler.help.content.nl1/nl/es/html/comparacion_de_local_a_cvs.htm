Comparación de local a cvs

A continuación se detallan los pasos a seguir para comparar un proyecto de modelado local con el que se encuentre en el CVS.

 

 

Pasos

1.   Sitúese en el Navegador Data Modeler, seleccione el proyecto, haga clic con el botón derecho del ratón y elija Comparar>Comparar con otro proyecto CVS.

 

 

2.   Se muestra una ventana en la que debe indicar la rama o la versión donde se encuentra el proyecto de modelado con el que se quiere comparar. A continuación haga clic en .

 

 

3.   Se comparan los elementos del modelo, incluyendo esquemas y tablas externas, y se abre una vista que muestra, en su parte superior, un árbol con las diferencias existentes entre las estructuras de los elementos seleccionados.

Los elementos que aparecen en el árbol y que contenga diferencias respecto al proyecto comparado, muestran un literal con el número de cambios que ese elemento tiene respecto al proyecto comparado.

En la parte inferior se muestran dos secciones, una sección muestra el contenido del elemento en el primer recurso seleccionado y la otra sección muestra el contenido del segundo, reflejando las diferencias que existen entre ambos. Las diferencias que aparecen resaltadas son los cambios entre el elemento origen y el elemento heredado común.

 

 

Sobre las diferencias mostradas en la parte inferior del editor, puede copiar la información desde CVS hacia local. Para más información, consulte: Merge para la importación del modelo
