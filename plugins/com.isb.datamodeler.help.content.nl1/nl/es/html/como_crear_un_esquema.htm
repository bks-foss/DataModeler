Cómo crear un Esquema

Un esquema es una colección de descriptores (tablas, vistas, secuencias, rutinas, etc.) que contendrán los diferentes objetos del modelo. Cada esquema debe pertenecer obligatoriamente a una aplicación y subaplicación.

 

Pasos

 

1.    Para crear una nuevo esquema, dentro de la vista Navegador Data Modeler, haga clic en el botón derecho, Nuevo>Esquema.

 

 

2. Aparece un asistente que le guía en la creación del esquema. La ventana del asistente contiene los siguientes campos:

-     Proyecto: Introduzca el nombre del proyecto asociado a la aplicación, por defecto se muestra el nombre del proyecto sobre el que esté situado. Para seleccionar un proyecto que ya exista, haga clic en el botón .

-      Nombre: Teclee un nombre para denominar el esquema. No pueden existir dos esquemas con el mismo nombre en el mismo modelo.

-      Descripción: Incluya una descripción.

Por último, haga clic en el botón  .

 

 

3.   En la vista Data Modeler se muestra el nuevo esquema que acaba de crear.

 

 

4.   Situado en el esquema, y haciendo clic en el botón derecho del ratón, aparece un menú contextual con las siguientes acciones:

-      Nuevo>Diagrama: El diagrama se utiliza para representar el modelo relacional correspondiente a los modelos de datos de las aplicaciones.

Consulte: Cómo crear un Diagrama

-     Nuevo>Tabla: Una tabla es una colección ordenada de una o más columnas de una colección no ordenada de registros. Suele representar una entidad.

Consulte: Cómo crear una Tabla

-     Eliminar del Modelo: Permite borrar el esquema de modelado.

-     Deseleccionar todo: Permite deseleccionar de la vista el/los proyecto/s activos en ese momento.

-     Por último, se mostrarán los proyectos existentes en el área de trabajo y que pueden ser seleccionados. Marcar o desmarcar el proyecto supondrá incluirlo o no en la vista. Además  si deselecciona un proyecto de modelado se cerrarán los editores asociados que estén abiertos.
