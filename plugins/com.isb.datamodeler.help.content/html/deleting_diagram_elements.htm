Deleting Diagram Elements

To delete an element included in the diagram, right click the element to display a popup menu with the following delete options:

•       The element is deleted from the diagram. This option is only available from the diagram and for the Table elements.

•      The element and its relationships (inbound and outbound) are deleted from the model and from all the diagrams in which they are drawn.
