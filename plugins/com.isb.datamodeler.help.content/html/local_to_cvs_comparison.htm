Local to CVS Comparison

The steps for comparing a local modelling project with a project in the CVS are shown below.

 

 

Steps

1.    Go to the Data Modeler Navigator, select the project, right click and select Compare with>Compare with Other CVS Project.

 

 

2.    This opens a window in which to specify the branch or version containing the modelling project with which to compare. Next, click .

 

 

3.    The elements of the model are compared, including schemas and external tables, and a view opens displaying, at the top, a tree with the differences existing between the structures of the selected element.

The elements that appear in the tree and which contain differences with the compared project display a literal with the number of changes in that element versus that of the compared project.

Two sections appear at the bottom of the comparator. One section displays the content of the element in the first selected resource and the other one displays the content of the second one, reflecting the differences that exist between both. The differences that appear highlighted are the changes between the source element and the common inherited element.

 

 

Regarding the differences that appear at the bottom of the editor, the information can be copied from CVS to local. For more information, see: Merge for Model Import
