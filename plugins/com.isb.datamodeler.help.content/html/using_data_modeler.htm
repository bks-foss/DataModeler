Using Data Modeler

Purpose

Data Modeler is a perspective integrated in the BKS Platform for creating data models using a relational model.

The tool consists primarily of the following views:

•     Data Modeler Navigator

•     Properties

 

In addition to a series of editors for modelling the applications.

The following figure displays the Data Modeler perspective with an open modelling project:

 

 

The elements automatically or manually generated with the Data Modeler must comply with the naming conventions specified document Data Modeler Naming Convention

 

Steps

1.   Start the Data Modeler perspective from the Helios platform. From the toolbar, select: Window>Open Perspective>Others..., and select Data Modeler.

 

 

The perspective opens, displaying a series of views and editors that will help to complete the functionality for the different modelling elements making up the applications.

2.   From the Data Modeler Navigator View, create a Data Modeler project or select a modelling project that already exists in your workspace. The Data Modeler project contains the schemas and modelling diagrams for the application.

See: Creating a Data Modeler Project
