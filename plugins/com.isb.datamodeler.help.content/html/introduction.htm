Introduction

Data Modeler is a perspective integrated in the BKS Platform for creating data models using a relational model.

This tool has been developed with the aim of shortening the lifecycle of applications by following the strategy of integrating all the tools under a single development environment.

 

With the creation of the Data Modeler for generating the data models, the following is achieved:

1.    The BKS logical model and data model are integrated in the same platform.

2.    Importing the data model physical tables from the data dictionary.
