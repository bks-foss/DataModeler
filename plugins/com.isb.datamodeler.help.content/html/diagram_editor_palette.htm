Diagram Editor Palette

To include elements by dragging them from the palette to the editor. It is usually available on the right-hand side of the editor, although it can be moved to the left.

The palette consists of the following elements:

•        If you select this element and, then, click the editor, a new table-type element will be created. For more information, see Creating a Table

•      If you select this element and, then, click inside a table, a new column element will be created inside the table. For more information, see Creating a Column

•     If you select this element and, then, click inside a table, a new Unique Key element will be created inside the table. For more information, see Creating a Unique Key.

•       Type of relationship set up between tables, by which the list of involved columns from the child table are part of its Primary Key (PK). For more information, see Diagram Relationships

•      Type of relationship that is established between tables, by which the list of involved columns from the child table are not part of its Primary Key (PK). For more information, see Diagram Relationships.

 

Furthermore, it contains the following buttons:

•      To hide the palette.

•       To set the mouse in selection mode.

•       To zoom in on an image in the editor. Whenever you left click the editor, the image is zoomed in.

•        To zoom out the image in the editor. Whenever you left click the editor, the image is zoomed out.

•       The following three types of notes can be linked to certain elements included in the editor:

- Note: To include a simple note.

- Text: To include a text box.

- Attached Note: To attach a note.

If you are positioned on a table, the floating palette can be used to perform the following actions:

•     Add a Column

•    Add a Unique Key
