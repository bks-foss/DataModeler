Data Model Comparator

The data model comparator can be used to compare functionally the elements of the modelling project, either local-to-local or local to CVS, identifying the differences in model, schema and external table elements.

The comparator is divided into three sections. The top part displays a tree with the differences that exist between the structures of the selected elements. The correspondences between elements are displayed by name and the different schemas hang from the root. The associated elements (tables) are displayed under the schemas and columns and restrictions are displayed under the tables. Additionally, the Attributes node is displayed under the elements, meaning that a difference exists in the properties of the corresponding element.

Two sections appear at the bottom of the comparator. One section displays the content of the element in the first selected resource and the other one displays the content of the second one, reflecting the differences that exist between both. The differences that appear highlighted are the changes between the source element and the common inherited element.
