Data Modeler Navigator

The Data Modeler Navigator view is used for creating or editing a data modeler project consisting of a Schema, Diagrams, Tables, etc. Initially, this view does not contain any project.

The tables can be dragged from the Data Modeler Navigator to the diagrams.

The following figure displays an example of a deployed project:

 

 

In the Data Modeler Navigator, right click to display the following actions among others:

•     Copy, Paste and Move Elements

•     Rename Elements

•     Import Tables

•     Detect Possible Relationships
