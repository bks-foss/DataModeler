Creating a Data Modeler Project

The Data Modeler project will contain the schemas, tables and other elements needed for defining the data model.

 

 

Steps

1.    To create a new project, in the Data Modeler Navigator view, right click and select New>Data Modeler Project.

 

 

2.    A wizard appears that guides you through the project creation process. It contains the following fields:

-      Application Acronym: String used for identifying the Application. It must contain 6 characters.

-      Subapplication Code: Numeric Code used for identifying the subapplication.

-      Project Name: It is automatically completed based on the acronym.  The suffix '_MD' is added to it.

-      Database Type: Currently, the only type of Database available is he New Data Dictionary.

-      Description: include a description for the project.

 Finally, click .

 

 

3.    The Data Modeler Navigator view displays the new project you have just created.

 

 

4.   Once the project has been created, in the Data Modeler view, go to the project and right click. A contextual menu appears displaying the following actions:

-      New>Data Modeler Project: To create a new modelling project.

-      New>Schema: To create a new schema.

See: Creating a Schema

-      Delete from Model: To delete the active project from model.

-       Validate Model: This provides validation to help detect errors in the modelling.

See: Model Validations

-      Compare> With Other CVS Project: to compare a local project with another one located in the CVS.

See: Local to CVS Comparison

-      Compare>With Other Local Project: to compare two locally selected modelling projects

See: Local to Local Comparison

-       Deselect All: To deselect the modelling project(s) currently active from the view.

-       Lastly, the existing projects will be displayed in the workspace, where they can be selected. Marking or unmarking the project will involve either including or not including it in the view. In addition, if a modelling project is deselected, any associated editors that may be open will be closed.
