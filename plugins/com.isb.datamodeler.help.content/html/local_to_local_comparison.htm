Local to Local Comparison

The steps to complete to compare two local modelling projects selected are shown below.

 

 

Steps

1.    Go to the Data Modeler Navigator, select the project, right click and select Compare>Compare with Other Local Project.

 

 

2.    This opens a window for specifying the project with which to compare.

Next, click .

 

 

If open comparison editors exist, a window informing of it will be displayed, providing the option to close them and continue with the comparison.

 

 

3.    The elements of the model are compared, including schemas and external tables, and a view opens displaying, at the top, a tree with the differences existing between the structures of the selected elements.

The elements that appear in the tree and which contain differences with the compared project display a literal with the number of changes in that element versus that of the compared project.

Two sections appear at the bottom of the comparator. One section displays the content of the element in the first selected resource and the other one displays the content of the second one, reflecting the differences that exist between both. The differences that appear highlighted are the changes between the source element and the common inherited element.

 

 

Regarding the differences that appear at the bottom of the editor, it will be possible to merge the compared information. For more information, see: Merge for Model Import
