Rename Elements

Purpose

The Data Modeler can be used to rename its schemas and tables automatically.

 

 

Steps

1.    In the Data Modeler Navigator, go to the element (schema or table) to be renamed. Right click and, in the popup menu, select Rename.

 

 

2.    A popup window appears where a new name can be entered for the selected element. Type a new name and click .

 

 

3.    Finally, the Data Modeler Navigator will display the element already renamed. If the renamed element is a schema, its corresponding diagrams will be renamed.
